Clash of Clans DataBase

  • We selected Clash of clans game to be our miniworld
  • We analysed the miniworld and found the entities and relation possible
  • We constructed the ER model and convertedd it into relational model
  • Further we normalised the relational model into 3NF
  • Finally, we created the database, populated it
  • To Query the database, a python scrpit was made
